# hipicon-designer-panel
It is a panel( admin dashboard ) for designers of hipicon.

# Features
- Dynamic Data Tables
- Graphs
- Dashboard
- Pagination
- Dynamic Modal for detail page
- Responsive

# What I Used ?
- React
- Json-server
- HTML
- SASS
- React-table
- Redux
- Recharts
- Bootstrap
